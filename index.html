Proyectos

ML WebApp - Calculadora Precio Inmuebles BOG

Proyecto en el que puse en práctica mis conocimientos en Analitica de datos y MachineLearning al utilizar datos tomados de una base de datos de OpenData del gobierno colombiano y cruzarla con una del DANE para obtener datos importantes con los cuales entrené un modelo de Regresión Linear para así crear una aplicación web que pudiera predecir los valores de un inmueble en la ciudad de Bogotá al alimentarlo con datos como la Zona donde se encuentra el inmueble, número de habitaciones, número de baños y área en metros cuadrados del inmueble.
La aplicación web fue creada utilizando el framework Flask en Python y para el entrenamiento del modelo utilicé las librerias Pandas, SciKit-learn y numpy en Python. La interfaz fue creada con HTML y la librería BootStrap 5.

Tecnologías Utilizadas: Python - Flask - SciKitLearn - Numpy - HTML - CSS - BootStrap5

Analisis de datos - Ecommerce

Proyecto en el cual se ponen en práctica las técnicas de analisis de datos para conocer el comportamiento de los clientes de una tienda E-commerce con el fin de ayudarlos a tomar una decisión a cerca de en que plataforma invertir capital para actualización.
Para este estudio se utilizaron las librerias de Pandas, SciKitLearn y numpy en Python.

Tecnologías Utilizadas: Python - SciKitLearn - Numpy
